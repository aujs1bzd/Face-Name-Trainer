# Face Name Trainer

Face Name Trainerは、顔写真を見ながら名前を回答することで、顔と名前の紐づけを練習するローカル完結型のReact/Viteアプリです。

## 主な機能

- 顔写真、姓、名、表示名、ふりがな、所属・チーム、メモ、別名・許容回答の登録、編集、削除
- 自由入力クイズと4択クイズ
- 表記ゆれを吸収する回答判定（空白、大小文字、ひらがな/カタカナ、敬称など）
- 間隔反復に基づく次回復習予定と、間違えた人物を優先しやすい出題
- 登録人数、今日復習すべき人数、総回答数、正答率、苦手な人物を表示するダッシュボード
- 名前・所属検索付き人物一覧
- JSONエクスポート / インポート
- 確認ダイアログ付き全削除

## セットアップ方法

```bash
npm install
```

## 起動方法

```bash
npm run dev
```

ブラウザでViteが表示するローカルURLを開いて利用してください。

## テスト

```bash
npm test
```

## データ保存とプライバシー

このアプリは、人物データと画像をブラウザのIndexedDBに保存します。外部バックエンド、外部API、クラウド同期、解析ツール、広告タグは使用していません。登録した顔写真や名前はサーバーへ送信されず、利用中のブラウザ内に保存されます。

ただし、顔写真・氏名・所属・メモは個人情報です。本人の同意や所属組織のルールに従って利用してください。共有PCや管理外端末では、利用後に必要に応じてエクスポート、削除、ブラウザデータ消去を行ってください。
